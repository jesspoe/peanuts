# README
Peanuts website 

* Ruby version 2.6.0 

* System dependencie: 
  gem 'bcrypt', '~> 3.1.7', 
  gem 'rspec-rails'

* Database creation: 
  rails db:create, 
  rails db:migrate
  
* How to run the test suite: 
  rspec 


* This website is built using Ruby on Rails - deployable on your local machine. 
* Purpose to showcase custom routes, MVC architectural pattern, security, and form creation.
